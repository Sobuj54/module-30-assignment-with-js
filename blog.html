1. What is DOM and what is the purpose of DOM?

Definition of DOM

DOM means Document Object Model. DOM is a programming interface for documents. This document enables Javascript to access and manipulate the elements and styles of a website. Purpose of DOM DOM is developing for maintaining a standard programming interface for HTML document. It can change the content of html like attribute, tag, id, css style. We can connect html file and javascript by using dom. We use DOM for changing dynamically in html page.

2. How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML element By id attribute value, name attribute by value, by tag, class or css class name.

Name the DOM methods

getElementsByTagName() getElementsByClassName() getElementById() getElementByName() querySelector() querySelectorAll()

3. What is bubble event?

Event Bubbling happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.
